# Forward Motion

*Each sentence must add new information to the sentence before it. Restatement, reinforcement, and explanation of a point already made are violations.*

## The Principle

A paragraph is a chain of information. Every link must add a new claim, a new detail, a new example, or a new turn. If sentence N+1 could be deleted and the reader would lose nothing, sentence N+1 is a violation.

This is the rule that prevents the "Bitcoin paragraph" failure — where a strong sentence is followed by three weaker sentences that say the same thing in different words.

## The Test

For every pair of adjacent sentences in the draft, ask this question:

**Does sentence N+1 introduce at least one of the following that is NOT present in sentence N?**

- A new claim (a new assertion about the world)
- A new entity (a named person, place, tool, number, or concrete object)
- A new consequence (a cause, effect, or implication of N)
- A new contrast (a turn, exception, or counterpoint to N)
- A new example (a specific instance that illustrates N)
- A new instruction (a next action for the reader)

If sentence N+1 does none of these, it is a violation.

## Common Violation Patterns

These are the structural forms violations usually take. Detecting any of these patterns triggers the fix:

1. **Semantic equivalence.** N+1 expresses the same proposition as N using different words. Example: "The system is valuable. It has a lot of worth." — both sentences assert the same thing.

2. **Reinforcement.** N+1 agrees with N without adding anything. Example: "This changes everything. It really does change everything."

3. **Explanation of the obvious.** N+1 explains a point from N that the reader has already understood. Example: "Getting serious with Claude in 2026 is like buying Bitcoin in 2010. The people who start now will have an advantage over those who wait." — the Bitcoin comparison already communicated the advantage; N+1 restates it.

4. **Hedged repetition.** N+1 softens or qualifies N without adding substance. Example: "You should do this today. You might want to consider starting soon."

5. **Transitional filler.** N+1 is a connector sentence that carries no new information. Example: "And that brings me to my next point."

## The Fix

When sentence N+1 is found to be a violation:

**First action: delete N+1.** Read the paragraph without it. If the paragraph still works, the deletion is complete.

**Second action: if deletion creates a logical gap, merge N+1's function into N.** Example: if N+1 was providing a weak example, strengthen N itself with that example.

**Third action: if the paragraph requires a bridge to the next idea, replace N+1 with a sentence that introduces the next idea, not one that restates the previous one.**

## The Exception

One class of exception is allowed: the standalone landing.

A short sentence that isolates a single strong claim (5 words or fewer) may follow a longer sentence that made the same claim, IF the standalone sentence serves as emphasis rather than explanation. Example: "The system runs itself. Period."

This exception is narrow. It applies only when:
- The standalone sentence is 5 words or fewer
- It adds emphasis through compression, not restatement through elaboration
- It appears at the end of a paragraph, not mid-paragraph

Three or more "emphasis" sentences stacked together is still a violation. Emphasis is singular by design.

## Detection Priority

Over-explanation is the highest-frequency failure in long-form writing. In any draft, apply this rule FIRST, before any other principle. Most bloat is forward-motion violations wearing other disguises.
