# Specificity

*Abstract nouns, vague quantifiers, and unnamed entities must be replaced with concrete, countable, or named references. Generality is a violation.*

## The Principle

Text contains two kinds of references: specific and abstract. Specific references name a countable, locatable, or concretely describable entity. Abstract references gesture at a category without naming a member.

Abstract references are detectable patterns in the text. They are not a matter of tone or feel — they are specific word categories that can be identified directly.

## Detectable Categories of Abstract Reference

These are the word and phrase classes that indicate abstraction. Each occurrence is a trigger for the fix.

**Vague quantifiers.** Words that reference quantity without specifying it:
- "a lot," "lots," "many," "most," "some," "several," "a few," "various," "numerous," "plenty," "a number of"

**Vague intensifiers.** Words that claim magnitude without measurement:
- "very," "really," "quite," "extremely," "tremendously," "hugely," "massively"

**Vague time references.** Phrases that reference time without naming it:
- "a while ago," "recently," "in the past," "these days," "nowadays," "at some point," "someday"

**Vague entity references.** Nouns that reference something without identifying it:
- "people," "things," "stuff," "something," "someone," "somewhere," "everyone," "anyone"

**Vague quality claims.** Adjectives that assert quality without evidence:
- "great," "amazing," "incredible," "powerful," "revolutionary," "transformative," "game-changing"

**Abstract nouns without referent.** Category words that do not name a specific instance:
- "success," "growth," "value," "transformation," "results," "impact," "potential"

## The Test

For every sentence in the draft, scan for occurrences of the above categories.

**VIOLATION if a sentence contains a vague quantifier and does not elsewhere specify the quantity in numerical or named form.**

**VIOLATION if a sentence contains a vague entity reference ("people," "things") without specifying which people or things within the same sentence or adjacent context.**

**VIOLATION if a sentence uses a vague quality claim ("great," "amazing") without supporting evidence in adjacent sentences.**

**VIOLATION if a sentence contains multiple abstract nouns and no concrete nouns or numbers.**

## The Fix

When a violation is detected, apply one of these fixes:

**Fix A: Replace with a number.** "A lot of operators" → "23 operators" or "most operators I work with" → "four out of five operators I've trained."

**Fix B: Replace with a named entity.** "A client" → "a client running a supplement brand" or "someone I know" → "my friend Sarah."

**Fix C: Replace with a concrete example.** "Things got better" → "revenue doubled and response times dropped."

**Fix D: Replace with a sensory-neutral specific.** Not a sensory description (which requires human perception) but a factual specific. "It was powerful" → "it cut the task from 3 hours to 20 minutes."

**Fix E: Delete if no specific can be supplied.** If the writer has no specific number, name, or example to substitute, the sentence is making an empty claim. Cut it.

## Strong Replacements to Prefer

Specificity has a hierarchy. From weakest to strongest:

- Abstract category ("success") → weakest
- Vague quantifier ("many people") → weak
- Approximate number ("about a hundred") → medium
- Exact number ("127") → strong
- Named entity with number ("127 subscribers on May 3") → strongest

When executing a fix, prefer the strongest available replacement the source material supports. Do not invent numbers or names that were not in the source.

## The Exception

Some abstractions are load-bearing and cannot be replaced. These include:
- Terms of art from the domain (e.g., "leverage," "positioning," "cascade" in marketing contexts)
- Abstract nouns used as proper labels (e.g., "Rampant Growth")
- Philosophical or conceptual discussions where abstraction is the subject

When an abstract reference is load-bearing, leave it. When it is filler, replace it. Distinguish by asking: is this word the subject being discussed, or is it standing in for something the writer failed to name? Only the latter is a violation.
