# Conor's Claude Skills & Agents

A public library of the skills I use to run my business through Claude. Free to install and use. I add to this and improve it over time, so check back.

A "skill" is a saved playbook you install once, and Claude follows it every time you ask. These are the writing and editing ones I'm comfortable sharing publicly.

## What's in here right now

**Humanizer** — erases the AI "slop" patterns from everything Claude writes for you. Em dashes, the "it's not just X, it's Y" reframe, rhythmic triplets, all the tells that make writing read as AI. Install it once and Claude checks its own writing against the rules before you ever see a draft.

**Coarse Sandpaper** — a second-pass editor for any long-form writing over 1,000 words. It tightens a draft that's already structured: cuts bloat, kills restatement, sharpens vague lines into specific ones. Run it after you've got a draft, before you polish the voice.

## How to install (about 30 seconds)

Your Claude can pull these straight from this repo. No downloading files.

1. In Claude, open the plugins menu and choose **Add marketplace**.
2. Paste this repo: `Conor-Boyland-ai/claude-skills-and-agents`
3. Browse the skills and install the ones you want.

When I update a skill, reinstall it to get the latest version.
